# 🍔 Food Website

A modern and responsive food ordering website built to provide users with an attractive interface for exploring menus and ordering food online.

## 🚀 Features

* Responsive and user-friendly design
* Beautiful homepage with featured food items
* Food menu with categories
* Search and navigation functionality
* Add-to-cart interface
* Contact and About pages
* Fast and clean UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap (if used)

## 📂 Project Structure

```
food_website/
│── index.html
│── css/
│── js/
│── images/
│── assets/
│── README.md
```

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/23it029-shyam/food_website.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
